### コンテナ情報表示
$ docker ps -a

### コンテナ保存
$ docker commit [CONTAINER ID] [リポジトリ:タグ]

### 保存イメージの確認
$ docker images

### コンテナ起動
$ docker run -i -t [リポジトリ:タグ]
